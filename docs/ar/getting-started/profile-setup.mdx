## إضافة ملف شخصي

1. افتح عرض **الملفات الشخصية** في الشريط الجانبي لـ F5 XC.
2. انقر على أيقونة **+** لإضافة ملف شخصي جديد.
3. أدخل عنوان URL الخاص بالمستأجر (مثال: `https://your-tenant.console.ves.volterra.io`).
4. اختر طريقة المصادقة:
   - **رمز API**: الصق رمز API الخاص بك من وحدة تحكم F5 XC.
   - **شهادة P12**: حدد ملف الشهادة `.p12` الخاص بك وأدخل كلمة المرور.
5. قم بتعيين مساحة اسم افتراضية.

## إدارة الملفات الشخصية

- **التبديل بين الملفات الشخصية**: انقر على ملف شخصي في عرض الملفات الشخصية لجعله نشطاً.
- **التعديل**: انقر بزر الماوس الأيمن على ملف شخصي واختر **تعديل الملف الشخصي**.
- **الحذف**: انقر بزر الماوس الأيمن على ملف شخصي واختر **حذف الملف الشخصي**.

يتم تخزين الملفات الشخصية بشكل آمن باستخدام التخزين السري المدمج في VS Code (متوافق مع XDG على Linux).
