## سوق VS Code

ابحث عن "F5 Distributed Cloud Tools" في لوحة الإضافات في VS Code (`Ctrl+Shift+X` / `Cmd+Shift+X`) وانقر على **تثبيت**.

## التثبيت اليدوي لملف VSIX

1. قم بتنزيل ملف `.vsix` من صفحة [إصدارات GitHub](https://github.com/f5xc-salesdemos/vscode-f5xc-tools/releases).
2. في VS Code، افتح لوحة الأوامر (`Ctrl+Shift+P` / `Cmd+Shift+P`).
3. قم بتشغيل **Extensions: Install from VSIX** وحدد الملف الذي تم تنزيله.

## المتطلبات

- VS Code الإصدار 1.107.0 أو أحدث
- مستأجر F5 Distributed Cloud
- بيانات اعتماد API (شهادة أو رمز API)
