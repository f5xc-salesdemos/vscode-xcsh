## مراقبة الحالة

تعرض طريقة عرض الشجرة **حالة السحابة** الصحة الحالية لخدمات F5 Distributed Cloud. يتم جلب الحالة من صفحة حالة F5 الرسمية وتعرض:

- الحالة العامة للمنصة
- صحة كل مكوّن (API، وحدة التحكم، مستوى البيانات، إلخ.)
- الحوادث النشطة ونوافذ الصيانة

يتم تحديث العرض تلقائياً بناءً على فترة التحديث المُعدَّة (`f5xc.autoRefreshInterval`).
