تدعم الإضافة أكثر من 236 نوعًا من الموارد عبر 25 نطاقًا من نطاقات API الخاصة بـ F5 XC. يتم تنظيم الموارد حسب فئة النطاق في عرض الشجرة.

## فئات النطاقات

- **تسليم التطبيقات**: HTTP Load Balancers، TCP Load Balancers، Origin Pools، Health Checks
- **الأمان**: Web App Firewalls، API Security، Bot Defense، DDoS Protection
- **الشبكات**: Virtual Networks، Sites، Fleet Management، Network Policies
- **نظام أسماء النطاقات**: DNS Zones، DNS Load Balancers، DNS Records
- **المراقبة**: Dashboards، Alerts، Log Receivers
- **البنية التحتية**: Cloud Sites، Cloud Credentials، Service Policies

يتم إنشاء القائمة الكاملة لأنواع الموارد المدعومة تلقائيًا من مواصفات OpenAPI الخاصة بـ F5 XC ويتم تحديثها تلقائيًا عند إصدار إصدارات جديدة من API.
