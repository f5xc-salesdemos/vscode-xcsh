أدوات F5 Distributed Cloud هي إضافة لـ VS Code تتيح لك تصفح وإنشاء وتحرير وحذف موارد F5 XC مباشرة من المحرر الخاص بك. توفر عمليات CRUD كاملة، وIntelliSense لـ JSON مع التحقق من صحة المخططات، وأدوات المراقبة، ومتابعة حالة السحابة لأكثر من 236 نوعًا من الموارد عبر 25 نطاقًا من واجهات API.
