## Validation de schéma

L'extension fournit des schémas JSON pour les types de ressources courants. Lors de l'édition de fichiers JSON de ressources, VS Code valide votre configuration en temps réel et met en évidence les erreurs.

## Schémas pris en charge

- HTTP Load Balancer (`.http_loadbalancer.json`)
- Origin Pool (`.origin_pool.json`)
- App Firewall / WAF (`.app_firewall.json`)
- Health Check (`.healthcheck.json`)
- TCP Load Balancer (`.tcp_loadbalancer.json`)
- DNS Load Balancer (`.dns_load_balancer.json`)
- Virtual Site (`.virtual_site.json`)

## Auto-complétion

IntelliSense fournit l'auto-complétion pour les noms de champs, les valeurs d'énumération et les structures d'objets imbriqués basées sur la spécification OpenAPI de F5 XC.
