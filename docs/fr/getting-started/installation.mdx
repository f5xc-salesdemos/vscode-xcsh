## Place de marché VS Code

Recherchez « F5 Distributed Cloud Tools » dans le panneau Extensions de VS Code (`Ctrl+Shift+X` / `Cmd+Shift+X`) et cliquez sur **Installer**.

## Installation manuelle VSIX

1. Téléchargez le fichier `.vsix` depuis la page [GitHub Releases](https://github.com/f5-sales-demo/vscode-f5xc-tools/releases).
2. Dans VS Code, ouvrez la palette de commandes (`Ctrl+Shift+P` / `Cmd+Shift+P`).
3. Exécutez **Extensions: Install from VSIX** et sélectionnez le fichier téléchargé.

## Prérequis

- VS Code 1.107.0 ou version ultérieure
- Un tenant F5 Distributed Cloud
- Des identifiants API (certificat ou jeton API)
