## Paramètres disponibles

| Paramètre | Type | Valeur par défaut | Description |
|---------|------|---------|-------------|
| `f5xc.logLevel` | string | `info` | Niveau de verbosité des journaux : `debug`, `info`, `warn`, `error` |
| `f5xc.defaultNamespace` | string | `default` | Espace de noms par défaut pour les opérations sur les ressources |
| `f5xc.confirmDelete` | boolean | `true` | Afficher une boîte de dialogue de confirmation avant la suppression de ressources |
| `f5xc.deleteConfirmationLevel` | string | `always` | Quand confirmer : `always`, `high-only`, `never` |
| `f5xc.autoRefreshInterval` | number | `0` | Intervalle d'actualisation automatique en secondes (0 = désactivé) |
| `f5xc.viewMode` | string | `console` | Mode d'affichage : `console` ou `full` |
| `f5xc.showPreviewApis` | boolean | `true` | Afficher les types de ressources d'API bêta/aperçu |
