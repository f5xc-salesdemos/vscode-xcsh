## Raccourcis clavier par défaut

Les raccourcis clavier peuvent être personnalisés via l'éditeur de raccourcis clavier de VS Code (`Ctrl+K Ctrl+S` / `Cmd+K Cmd+S`). Recherchez « f5xc » pour trouver toutes les commandes de l'extension.

Les plus de 40 commandes sont accessibles via la Palette de commandes (`Ctrl+Shift+P` / `Cmd+Shift+P`) en recherchant « F5 XC ».
