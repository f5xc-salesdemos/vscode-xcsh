F5 Distributed Cloud Tools est une extension VS Code qui vous permet de parcourir, créer, modifier et supprimer des ressources F5 XC directement depuis votre éditeur. Elle fournit des opérations CRUD complètes, l'IntelliSense JSON avec validation de schéma, des outils d'observabilité et la surveillance de l'état du cloud pour plus de 236 types de ressources répartis sur 25 domaines d'API.
