L'extension prend en charge plus de 236 types de ressources répartis sur 25 domaines d'API F5 XC. Les ressources sont organisées par catégorie de domaine dans l'arborescence.

## Catégories de domaines

- **Livraison d'applications** : HTTP Load Balancers, TCP Load Balancers, Origin Pools, Health Checks
- **Sécurité** : Web App Firewalls, API Security, Bot Defense, DDoS Protection
- **Réseau** : Virtual Networks, Sites, Fleet Management, Network Policies
- **DNS** : DNS Zones, DNS Load Balancers, DNS Records
- **Observabilité** : Dashboards, Alerts, Log Receivers
- **Infrastructure** : Cloud Sites, Cloud Credentials, Service Policies

La liste complète des types de ressources pris en charge est générée automatiquement à partir de la spécification OpenAPI de F5 XC et se met à jour automatiquement lorsque de nouvelles versions d'API sont publiées.
