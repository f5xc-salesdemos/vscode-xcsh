F5 Distributed Cloud 工具是一款 VS Code 扩展，使您能够直接从编辑器中浏览、创建、编辑和删除 F5 XC 资源。它提供完整的 CRUD 操作、带有架构验证的 JSON IntelliSense、可观测性工具以及云状态监控，支持跨 25 个 API 域的 236 多种资源类型。
