## 状态监控

**云端状态**树视图显示 F5 Distributed Cloud 服务的当前健康状况。状态信息从 F5 官方状态页面获取，并显示：

- 整体平台状态
- 各组件健康状况（API、控制台、数据平面等）
- 活跃的事件和维护窗口

该视图会根据配置的刷新间隔（`f5xc.autoRefreshInterval`）自动更新。
