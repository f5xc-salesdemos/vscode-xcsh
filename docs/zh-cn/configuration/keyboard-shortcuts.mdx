## 默认快捷键绑定

键盘快捷键可以通过 VS Code 的键盘快捷方式编辑器（`Ctrl+K Ctrl+S` / `Cmd+K Cmd+S`）进行自定义。搜索 "f5xc" 可查找所有扩展命令。

所有 40 多个命令均可通过命令面板（`Ctrl+Shift+P` / `Cmd+Shift+P`）搜索 "F5 XC" 来访问。
