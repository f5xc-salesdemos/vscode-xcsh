## 可用设置

| 设置 | 类型 | 默认值 | 描述 |
|---------|------|---------|-------------|
| `f5xc.logLevel` | string | `info` | 日志详细程度：`debug`、`info`、`warn`、`error` |
| `f5xc.defaultNamespace` | string | `default` | 资源操作的默认命名空间 |
| `f5xc.confirmDelete` | boolean | `true` | 删除资源前显示确认对话框 |
| `f5xc.deleteConfirmationLevel` | string | `always` | 确认时机：`always`、`high-only`、`never` |
| `f5xc.autoRefreshInterval` | number | `0` | 自动刷新间隔（秒）（0 = 禁用） |
| `f5xc.viewMode` | string | `console` | 视图模式：`console` 或 `full` |
| `f5xc.showPreviewApis` | boolean | `true` | 显示测试版/预览版 API 资源类型 |
