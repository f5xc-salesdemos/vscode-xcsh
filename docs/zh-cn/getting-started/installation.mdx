## VS Code 扩展商店

在 VS Code 扩展面板（`Ctrl+Shift+X` / `Cmd+Shift+X`）中搜索 "F5 Distributed Cloud Tools"，然后点击**安装**。

## 手动 VSIX 安装

1. 从 [GitHub Releases](https://github.com/f5-sales-demo/vscode-f5xc-tools/releases) 页面下载 `.vsix` 文件。
2. 在 VS Code 中，打开命令面板（`Ctrl+Shift+P` / `Cmd+Shift+P`）。
3. 运行 **Extensions: Install from VSIX** 并选择下载的文件。

## 系统要求

- VS Code 1.107.0 或更高版本
- 一个 F5 Distributed Cloud 租户
- API 凭据（证书或 API 令牌）
