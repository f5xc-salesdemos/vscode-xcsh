该扩展支持跨 25 个 F5 XC API 域的 236+ 种资源类型。资源在树视图中按域类别进行组织。

## 域类别

- **应用交付**：HTTP 负载均衡器、TCP 负载均衡器、源站池、健康检查
- **安全**：Web 应用防火墙、API 安全、Bot 防护、DDoS 防护
- **网络**：虚拟网络、站点、集群管理、网络策略
- **DNS**：DNS 区域、DNS 负载均衡器、DNS 记录
- **可观测性**：仪表板、告警、日志接收器
- **基础设施**：云站点、云凭证、服务策略

支持的资源类型完整列表由 F5 XC OpenAPI 规范自动生成，并在新 API 版本发布时自动更新。
