## การเพิ่มโปรไฟล์

1. เปิดมุมมอง **Profiles** ในแถบด้านข้างของ F5 XC
2. คลิกไอคอน **+** เพื่อเพิ่มโปรไฟล์ใหม่
3. ป้อน URL ของ tenant ของคุณ (เช่น `https://your-tenant.console.ves.volterra.io`)
4. เลือกวิธีการยืนยันตัวตน:
   - **API Token**: วาง API token ของคุณจากคอนโซล F5 XC
   - **P12 Certificate**: เลือกไฟล์ใบรับรอง `.p12` ของคุณและป้อนรหัสผ่าน
5. ตั้งค่า namespace เริ่มต้น

## การจัดการโปรไฟล์

- **สลับโปรไฟล์**: คลิกโปรไฟล์ในมุมมอง Profiles เพื่อทำให้เป็นโปรไฟล์ที่ใช้งานอยู่
- **แก้ไข**: คลิกขวาที่โปรไฟล์และเลือก **Edit Profile**
- **ลบ**: คลิกขวาที่โปรไฟล์และเลือก **Delete Profile**

โปรไฟล์จะถูกจัดเก็บอย่างปลอดภัยโดยใช้ที่เก็บข้อมูลลับในตัวของ VS Code (รองรับ XDG บน Linux)
