## VS Code Marketplace

ค้นหา "F5 Distributed Cloud Tools" ในแผง Extensions ของ VS Code (`Ctrl+Shift+X` / `Cmd+Shift+X`) แล้วคลิก **ติดตั้ง**

## การติดตั้ง VSIX ด้วยตนเอง

1. ดาวน์โหลดไฟล์ `.vsix` จากหน้า [GitHub Releases](https://github.com/f5-sales-demo/vscode-f5xc-tools/releases)
2. ใน VS Code ให้เปิด Command Palette (`Ctrl+Shift+P` / `Cmd+Shift+P`)
3. เรียกใช้ **Extensions: Install from VSIX** แล้วเลือกไฟล์ที่ดาวน์โหลดมา

## ข้อกำหนด

- VS Code 1.107.0 หรือใหม่กว่า
- เทแนนต์ F5 Distributed Cloud
- ข้อมูลรับรอง API (ใบรับรองหรือโทเค็น API)
