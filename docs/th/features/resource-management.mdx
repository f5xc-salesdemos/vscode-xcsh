## การเรียกดูทรัพยากร

มุมมองแบบต้นไม้ **Resources** จะแสดงทรัพยากรที่จัดระเบียบตามโดเมนและหมวดหมู่ ขยายหมวดหมู่เพื่อดูประเภทของทรัพยากร จากนั้นขยายประเภทเพื่อแสดงรายการทรัพยากรแต่ละรายการใน namespace ของคุณ

## การสร้างทรัพยากร

1. คลิกขวาที่ประเภทของทรัพยากรในต้นไม้แล้วเลือก **Create**
2. ตัวแก้ไข JSON จะเปิดขึ้นพร้อมเทมเพลตที่มีการตรวจสอบ schema
3. กรอกฟิลด์ที่จำเป็น (IntelliSense จะช่วยเติมข้อมูลอัตโนมัติ)
4. บันทึกไฟล์เพื่อส่งทรัพยากรไปยัง F5 XC API

## การแก้ไขทรัพยากร

1. คลิกทรัพยากรในต้นไม้เพื่อเปิดในตัวแก้ไข
2. แก้ไขฟิลด์พร้อมการรองรับ IntelliSense อย่างเต็มรูปแบบ
3. บันทึกเพื่อส่งการเปลี่ยนแปลงไปยัง API

## การลบทรัพยากร

1. คลิกขวาที่ทรัพยากรแล้วเลือก **Delete**
2. ยืนยันการลบ (สามารถกำหนดค่าได้ผ่าน `f5xc.confirmDelete` และ `f5xc.deleteConfirmationLevel`)
