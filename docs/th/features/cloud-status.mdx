## การตรวจสอบสถานะ

มุมมองแบบต้นไม้ **Cloud Status** แสดงสุขภาพปัจจุบันของบริการ F5 Distributed Cloud ข้อมูลสถานะถูกดึงมาจากหน้าสถานะอย่างเป็นทางการของ F5 และแสดง:

- สถานะแพลตฟอร์มโดยรวม
- สุขภาพของแต่ละส่วนประกอบ (API, Console, Data Plane เป็นต้น)
- เหตุการณ์ที่กำลังดำเนินอยู่และช่วงเวลาการบำรุงรักษา

มุมมองนี้จะอัปเดตโดยอัตโนมัติตามช่วงเวลาการรีเฟรชที่กำหนดค่าไว้ (`f5xc.autoRefreshInterval`)
