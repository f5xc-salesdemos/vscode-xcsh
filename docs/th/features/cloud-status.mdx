## การตรวจสอบสถานะ

มุมมองแบบต้นไม้ **Cloud Status** จะแสดงสุขภาพปัจจุบันของบริการ F5 Distributed Cloud สถานะจะถูกดึงมาจากหน้าสถานะอย่างเป็นทางการของ F5 และแสดง:

- สถานะแพลตฟอร์มโดยรวม
- สุขภาพของแต่ละคอมโพเนนต์ (API, Console, Data Plane เป็นต้น)
- เหตุการณ์ที่เกิดขึ้นและช่วงเวลาบำรุงรักษา

มุมมองนี้จะอัปเดตโดยอัตโนมัติตามช่วงเวลาการรีเฟรชที่กำหนดค่าไว้ (`f5xc.autoRefreshInterval`)
