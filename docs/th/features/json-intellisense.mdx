## การตรวจสอบสคีมา

ส่วนขยายนี้มี JSON schema สำหรับประเภททรัพยากรทั่วไป เมื่อแก้ไขไฟล์ JSON ของทรัพยากร VS Code จะตรวจสอบการกำหนดค่าของคุณแบบเรียลไทม์และเน้นข้อผิดพลาด

## สคีมาที่รองรับ

- HTTP Load Balancer (`.http_loadbalancer.json`)
- Origin Pool (`.origin_pool.json`)
- App Firewall / WAF (`.app_firewall.json`)
- Health Check (`.healthcheck.json`)
- TCP Load Balancer (`.tcp_loadbalancer.json`)
- DNS Load Balancer (`.dns_load_balancer.json`)
- Virtual Site (`.virtual_site.json`)

## การเติมข้อความอัตโนมัติ

IntelliSense ให้การเติมข้อความอัตโนมัติสำหรับชื่อฟิลด์ ค่า enum และโครงสร้างอ็อบเจกต์ซ้อนกัน โดยอ้างอิงจากข้อกำหนด OpenAPI ของ F5 XC
