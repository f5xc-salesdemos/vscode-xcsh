ส่วนขยายนี้รองรับทรัพยากรมากกว่า 236 ประเภท ครอบคลุม 25 โดเมน API ของ F5 XC ทรัพยากรถูกจัดระเบียบตามหมวดหมู่โดเมนในมุมมองแบบต้นไม้

## หมวดหมู่โดเมน

- **การส่งมอบแอปพลิเคชัน**: HTTP Load Balancers, TCP Load Balancers, Origin Pools, Health Checks
- **ความปลอดภัย**: Web App Firewalls, API Security, Bot Defense, DDoS Protection
- **เครือข่าย**: Virtual Networks, Sites, Fleet Management, Network Policies
- **DNS**: DNS Zones, DNS Load Balancers, DNS Records
- **การสังเกตการณ์**: Dashboards, Alerts, Log Receivers
- **โครงสร้างพื้นฐาน**: Cloud Sites, Cloud Credentials, Service Policies

รายการทรัพยากรประเภทที่รองรับทั้งหมดถูกสร้างขึ้นโดยอัตโนมัติจากข้อกำหนด OpenAPI ของ F5 XC และจะอัปเดตโดยอัตโนมัติเมื่อมีการเผยแพร่เวอร์ชัน API ใหม่
