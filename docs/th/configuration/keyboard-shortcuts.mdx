## การกำหนดปุ่มลัดเริ่มต้น

ปุ่มลัดคีย์บอร์ดสามารถปรับแต่งได้ผ่านตัวแก้ไขปุ่มลัดคีย์บอร์ดของ VS Code (`Ctrl+K Ctrl+S` / `Cmd+K Cmd+S`) ค้นหา "f5xc" เพื่อค้นหาคำสั่งของส่วนขยายทั้งหมด

คำสั่งทั้งหมดกว่า 40 รายการสามารถเข้าถึงได้ผ่าน Command Palette (`Ctrl+Shift+P` / `Cmd+Shift+P`) โดยค้นหา "F5 XC"
