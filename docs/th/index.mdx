F5 Distributed Cloud Tools เป็นส่วนขยาย VS Code ที่ช่วยให้คุณสามารถเรียกดู สร้าง แก้ไข และลบทรัพยากร F5 XC ได้โดยตรงจากเอดิเตอร์ของคุณ โดยรองรับการดำเนินการ CRUD อย่างครบถ้วน JSON IntelliSense พร้อมการตรวจสอบสคีมา เครื่องมือสังเกตการณ์ระบบ และการตรวจสอบสถานะคลาวด์สำหรับทรัพยากรมากกว่า 236 ประเภท ครอบคลุม 25 โดเมน API
