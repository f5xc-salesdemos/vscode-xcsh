## Validazione dello schema

L'estensione fornisce schemi JSON per i tipi di risorse più comuni. Durante la modifica dei file JSON delle risorse, VS Code valida la configurazione in tempo reale ed evidenzia gli errori.

## Schemi supportati

- HTTP Load Balancer (`.http_loadbalancer.json`)
- Origin Pool (`.origin_pool.json`)
- App Firewall / WAF (`.app_firewall.json`)
- Health Check (`.healthcheck.json`)
- TCP Load Balancer (`.tcp_loadbalancer.json`)
- DNS Load Balancer (`.dns_load_balancer.json`)
- Virtual Site (`.virtual_site.json`)

## Completamento automatico

IntelliSense fornisce il completamento automatico per i nomi dei campi, i valori enum e le strutture degli oggetti annidati basandosi sulle specifiche OpenAPI di F5 XC.
