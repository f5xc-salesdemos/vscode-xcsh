## VS Code Marketplace

Cercare "F5 Distributed Cloud Tools" nel pannello Estensioni di VS Code (`Ctrl+Shift+X` / `Cmd+Shift+X`) e fare clic su **Installa**.

## Installazione manuale tramite VSIX

1. Scaricare il file `.vsix` dalla pagina [GitHub Releases](https://github.com/f5-sales-demo/vscode-f5xc-tools/releases).
2. In VS Code, aprire il Riquadro comandi (`Ctrl+Shift+P` / `Cmd+Shift+P`).
3. Eseguire **Extensions: Install from VSIX** e selezionare il file scaricato.

## Requisiti

- VS Code 1.107.0 o versioni successive
- Un tenant F5 Distributed Cloud
- Credenziali API (certificato o token API)
