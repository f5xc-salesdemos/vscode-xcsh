L'estensione supporta oltre 236 tipi di risorse distribuiti su 25 domini API di F5 XC. Le risorse sono organizzate per categoria di dominio nella vista ad albero.

## Categorie di dominio

- **Application Delivery**: HTTP Load Balancer, TCP Load Balancer, Origin Pool, Health Check
- **Sicurezza**: Web App Firewall, API Security, Bot Defense, Protezione DDoS
- **Rete**: Reti virtuali, Siti, Gestione della flotta, Policy di rete
- **DNS**: Zone DNS, DNS Load Balancer, Record DNS
- **Osservabilità**: Dashboard, Avvisi, Ricevitori di log
- **Infrastruttura**: Siti cloud, Credenziali cloud, Policy di servizio

L'elenco completo dei tipi di risorse supportati viene generato automaticamente dalla specifica OpenAPI di F5 XC e si aggiorna automaticamente quando vengono rilasciate nuove versioni delle API.
