## Associazioni Tasti Predefinite

Le scorciatoie da tastiera possono essere personalizzate tramite l'editor Scorciatoie da Tastiera di VS Code (`Ctrl+K Ctrl+S` / `Cmd+K Cmd+S`). Cerca "f5xc" per trovare tutti i comandi dell'estensione.

Tutti gli oltre 40 comandi sono accessibili tramite il Riquadro Comandi (`Ctrl+Shift+P` / `Cmd+Shift+P`) cercando "F5 XC".
