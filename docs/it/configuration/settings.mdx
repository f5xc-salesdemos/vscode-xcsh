## Impostazioni disponibili

| Impostazione | Tipo | Predefinito | Descrizione |
|---------|------|---------|-------------|
| `f5xc.logLevel` | string | `info` | Livello di dettaglio dei log: `debug`, `info`, `warn`, `error` |
| `f5xc.defaultNamespace` | string | `default` | Namespace predefinito per le operazioni sulle risorse |
| `f5xc.confirmDelete` | boolean | `true` | Mostra una finestra di conferma prima di eliminare le risorse |
| `f5xc.deleteConfirmationLevel` | string | `always` | Quando richiedere conferma: `always`, `high-only`, `never` |
| `f5xc.autoRefreshInterval` | number | `0` | Intervallo di aggiornamento automatico in secondi (0 = disabilitato) |
| `f5xc.viewMode` | string | `console` | Modalità di visualizzazione: `console` o `full` |
| `f5xc.showPreviewApis` | boolean | `true` | Mostra i tipi di risorse API in beta/anteprima |
