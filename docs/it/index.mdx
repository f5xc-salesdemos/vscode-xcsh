F5 Distributed Cloud Tools è un'estensione VS Code che consente di esplorare, creare, modificare ed eliminare le risorse F5 XC direttamente dal proprio editor. Fornisce operazioni CRUD complete, JSON IntelliSense con validazione degli schemi, strumenti di osservabilità e monitoraggio dello stato del cloud per oltre 236 tipi di risorse distribuiti su 25 domini API.
