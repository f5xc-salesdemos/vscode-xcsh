F5 Distributed Cloud Tools es una extensión de VS Code que le permite explorar, crear, editar y eliminar recursos de F5 XC directamente desde su editor. Proporciona operaciones CRUD completas, IntelliSense JSON con validación de esquemas, herramientas de observabilidad y monitoreo del estado de la nube para más de 236 tipos de recursos en 25 dominios de API.
