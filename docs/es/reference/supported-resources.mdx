La extensión soporta más de 236 tipos de recursos en 25 dominios de API de F5 XC. Los recursos están organizados por categoría de dominio en la vista de árbol.

## Categorías de Dominio

- **Entrega de Aplicaciones**: HTTP Load Balancers, TCP Load Balancers, Origin Pools, Health Checks
- **Seguridad**: Web App Firewalls, API Security, Bot Defense, DDoS Protection
- **Redes**: Virtual Networks, Sites, Fleet Management, Network Policies
- **DNS**: DNS Zones, DNS Load Balancers, DNS Records
- **Observabilidad**: Dashboards, Alerts, Log Receivers
- **Infraestructura**: Cloud Sites, Cloud Credentials, Service Policies

La lista completa de tipos de recursos soportados se genera automáticamente a partir de la especificación OpenAPI de F5 XC y se actualiza automáticamente cuando se publican nuevas versiones de la API.
