## Agregar un perfil

1. Abra la vista de **Perfiles** en la barra lateral de F5 XC.
2. Haga clic en el icono **+** para agregar un nuevo perfil.
3. Ingrese la URL de su tenant (por ejemplo, `https://your-tenant.console.ves.volterra.io`).
4. Elija un método de autenticación:
   - **Token de API**: Pegue su token de API desde la consola de F5 XC.
   - **Certificado P12**: Seleccione su archivo de certificado `.p12` e ingrese la contraseña.
5. Establezca un namespace predeterminado.

## Gestión de perfiles

- **Cambiar perfiles**: Haga clic en un perfil en la vista de Perfiles para activarlo.
- **Editar**: Haga clic derecho en un perfil y seleccione **Editar perfil**.
- **Eliminar**: Haga clic derecho en un perfil y seleccione **Eliminar perfil**.

Los perfiles se almacenan de forma segura utilizando el almacenamiento de secretos integrado de VS Code (compatible con XDG en Linux).
