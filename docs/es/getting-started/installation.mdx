## VS Code Marketplace

Busque "F5 Distributed Cloud Tools" en el panel de Extensiones de VS Code (`Ctrl+Shift+X` / `Cmd+Shift+X`) y haga clic en **Instalar**.

## Instalación manual de VSIX

1. Descargue el archivo `.vsix` desde la página de [GitHub Releases](https://github.com/f5-sales-demo/vscode-f5xc-tools/releases).
2. En VS Code, abra la Paleta de Comandos (`Ctrl+Shift+P` / `Cmd+Shift+P`).
3. Ejecute **Extensions: Install from VSIX** y seleccione el archivo descargado.

## Requisitos

- VS Code 1.107.0 o posterior
- Un tenant de F5 Distributed Cloud
- Credenciales de API (certificado o token de API)
