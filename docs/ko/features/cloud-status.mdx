## 상태 모니터링

**클라우드 상태** 트리 뷰는 F5 Distributed Cloud 서비스의 현재 상태를 표시합니다. 상태는 공식 F5 상태 페이지에서 가져오며 다음을 보여줍니다:

- 전체 플랫폼 상태
- 컴포넌트별 상태 (API, Console, Data Plane 등)
- 활성 인시던트 및 유지보수 일정

이 뷰는 설정된 새로고침 간격(`f5xc.autoRefreshInterval`)에 따라 자동으로 업데이트됩니다.
