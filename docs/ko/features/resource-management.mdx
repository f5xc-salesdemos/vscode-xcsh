## 리소스 탐색

**Resources** 트리 뷰는 도메인과 카테고리별로 구성된 리소스를 표시합니다. 카테고리를 확장하면 리소스 유형이 표시되며, 유형을 확장하면 네임스페이스 내의 개별 리소스 목록을 확인할 수 있습니다.

## 리소스 생성

1. 트리에서 리소스 유형을 마우스 오른쪽 버튼으로 클릭하고 **Create**를 선택합니다.
2. 스키마가 검증된 템플릿이 포함된 JSON 편집기가 열립니다.
3. 필수 필드를 입력합니다 (IntelliSense가 자동 완성을 제공합니다).
4. 파일을 저장하면 리소스가 F5 XC API로 제출됩니다.

## 리소스 편집

1. 트리에서 리소스를 클릭하여 편집기에서 엽니다.
2. 완전한 IntelliSense 지원을 통해 필드를 수정합니다.
3. 저장하면 변경 사항이 API로 푸시됩니다.

## 리소스 삭제

1. 리소스를 마우스 오른쪽 버튼으로 클릭하고 **Delete**를 선택합니다.
2. 삭제를 확인합니다 (`f5xc.confirmDelete` 및 `f5xc.deleteConfirmationLevel`을 통해 설정 가능).
