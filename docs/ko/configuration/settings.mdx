## 사용 가능한 설정

| 설정 | 타입 | 기본값 | 설명 |
|---------|------|---------|-------------|
| `f5xc.logLevel` | string | `info` | 로그 상세 수준: `debug`, `info`, `warn`, `error` |
| `f5xc.defaultNamespace` | string | `default` | 리소스 작업의 기본 네임스페이스 |
| `f5xc.confirmDelete` | boolean | `true` | 리소스 삭제 전 확인 대화 상자 표시 |
| `f5xc.deleteConfirmationLevel` | string | `always` | 확인 시점: `always`, `high-only`, `never` |
| `f5xc.autoRefreshInterval` | number | `0` | 자동 새로고침 간격(초 단위, 0 = 비활성화) |
| `f5xc.viewMode` | string | `console` | 보기 모드: `console` 또는 `full` |
| `f5xc.showPreviewApis` | boolean | `true` | 베타/미리보기 API 리소스 유형 표시 |
