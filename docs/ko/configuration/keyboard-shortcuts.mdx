## 기본 키 바인딩

키보드 단축키는 VS Code의 키보드 단축키 편집기(`Ctrl+K Ctrl+S` / `Cmd+K Cmd+S`)를 통해 사용자 정의할 수 있습니다. "f5xc"를 검색하면 모든 확장 명령을 찾을 수 있습니다.

40개 이상의 모든 명령은 명령 팔레트(`Ctrl+Shift+P` / `Cmd+Shift+P`)에서 "F5 XC"를 검색하여 접근할 수 있습니다.
