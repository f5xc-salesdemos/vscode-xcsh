## VS Code 마켓플레이스

VS Code 확장 프로그램 패널(`Ctrl+Shift+X` / `Cmd+Shift+X`)에서 "F5 Distributed Cloud Tools"를 검색하고 **설치**를 클릭합니다.

## 수동 VSIX 설치

1. [GitHub Releases](https://github.com/f5-sales-demo/vscode-f5xc-tools/releases) 페이지에서 `.vsix` 파일을 다운로드합니다.
2. VS Code에서 명령 팔레트(`Ctrl+Shift+P` / `Cmd+Shift+P`)를 엽니다.
3. **Extensions: Install from VSIX**를 실행하고 다운로드한 파일을 선택합니다.

## 요구 사항

- VS Code 1.107.0 이상
- F5 Distributed Cloud 테넌트
- API 자격 증명(인증서 또는 API 토큰)
