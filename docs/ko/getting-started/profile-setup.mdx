## 프로필 추가

1. F5 XC 사이드바에서 **Profiles** 뷰를 엽니다.
2. **+** 아이콘을 클릭하여 새 프로필을 추가합니다.
3. 테넌트 URL을 입력합니다 (예: `https://your-tenant.console.ves.volterra.io`).
4. 인증 방법을 선택합니다:
   - **API Token**: F5 XC 콘솔에서 발급받은 API 토큰을 붙여넣습니다.
   - **P12 Certificate**: `.p12` 인증서 파일을 선택하고 비밀번호를 입력합니다.
5. 기본 네임스페이스를 설정합니다.

## 프로필 관리

- **프로필 전환**: Profiles 뷰에서 프로필을 클릭하여 활성 프로필로 설정합니다.
- **편집**: 프로필을 마우스 오른쪽 버튼으로 클릭하고 **Edit Profile**을 선택합니다.
- **삭제**: 프로필을 마우스 오른쪽 버튼으로 클릭하고 **Delete Profile**을 선택합니다.

프로필은 VS Code의 내장 비밀 저장소(Linux에서는 XDG 호환)를 사용하여 안전하게 저장됩니다.
