이 확장 프로그램은 25개의 F5 XC API 도메인에 걸쳐 236개 이상의 리소스 유형을 지원합니다. 리소스는 트리 뷰에서 도메인 카테고리별로 구성됩니다.

## 도메인 카테고리

- **애플리케이션 전송**: HTTP 로드 밸런서, TCP 로드 밸런서, 오리진 풀, 헬스 체크
- **보안**: 웹 앱 방화벽, API 보안, 봇 방어, DDoS 보호
- **네트워킹**: 가상 네트워크, 사이트, 플릿 관리, 네트워크 정책
- **DNS**: DNS 존, DNS 로드 밸런서, DNS 레코드
- **관측성**: 대시보드, 알림, 로그 수신기
- **인프라**: 클라우드 사이트, 클라우드 자격 증명, 서비스 정책

지원되는 리소스 유형의 전체 목록은 F5 XC OpenAPI 사양에서 자동으로 생성되며, 새로운 API 버전이 출시될 때 자동으로 업데이트됩니다.
