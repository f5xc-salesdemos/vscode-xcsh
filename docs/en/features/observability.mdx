## Viewing Logs

Right-click a resource and select **View Logs** to stream real-time log data from the F5 XC platform.

## Viewing Metrics

Right-click a resource and select **View Metrics** to see performance and traffic metrics.

## Architecture Diagrams

Select **Diagram** on a resource to generate a visual architecture diagram showing the resource's relationships and dependencies.
