## Status Monitoring

The **Cloud Status** tree view displays the current health of F5 Distributed Cloud services. Status is fetched from the official F5 status page and shows:

- Overall platform status
- Per-component health (API, Console, Data Plane, etc.)
- Active incidents and maintenance windows

The view updates automatically based on the configured refresh interval (`f5xc.autoRefreshInterval`).
