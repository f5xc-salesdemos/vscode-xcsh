## VS Code Marketplace

Search for "F5 Distributed Cloud Tools" in the VS Code Extensions panel (`Ctrl+Shift+X` / `Cmd+Shift+X`) and click **Install**.

## Manual VSIX Install

1. Download the `.vsix` file from the [GitHub Releases](https://github.com/f5-sales-demo/vscode-f5xc-tools/releases) page.
2. In VS Code, open the Command Palette (`Ctrl+Shift+P` / `Cmd+Shift+P`).
3. Run **Extensions: Install from VSIX** and select the downloaded file.

## Requirements

- VS Code 1.107.0 or later
- An F5 Distributed Cloud tenant
- API credentials (certificate or API token)
