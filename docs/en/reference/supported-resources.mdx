The extension supports 236+ resource types across 25 F5 XC API domains. Resources are organized by domain category in the tree view.

## Domain Categories

- **Application Delivery**: HTTP Load Balancers, TCP Load Balancers, Origin Pools, Health Checks
- **Security**: Web App Firewalls, API Security, Bot Defense, DDoS Protection
- **Networking**: Virtual Networks, Sites, Fleet Management, Network Policies
- **DNS**: DNS Zones, DNS Load Balancers, DNS Records
- **Observability**: Dashboards, Alerts, Log Receivers
- **Infrastructure**: Cloud Sites, Cloud Credentials, Service Policies

The full list of supported resource types is auto-generated from the F5 XC OpenAPI specification and updates automatically when new API versions are released.
