## Default Keybindings

Keyboard shortcuts can be customized via VS Code's Keyboard Shortcuts editor (`Ctrl+K Ctrl+S` / `Cmd+K Cmd+S`). Search for "f5xc" to find all extension commands.

All 40+ commands are accessible via the Command Palette (`Ctrl+Shift+P` / `Cmd+Shift+P`) by searching for "F5 XC".
