## Configurações Disponíveis

| Configuração | Tipo | Padrão | Descrição |
|---------|------|---------|-------------|
| `f5xc.logLevel` | string | `info` | Verbosidade do log: `debug`, `info`, `warn`, `error` |
| `f5xc.defaultNamespace` | string | `default` | Namespace padrão para operações de recursos |
| `f5xc.confirmDelete` | boolean | `true` | Exibir diálogo de confirmação antes de excluir recursos |
| `f5xc.deleteConfirmationLevel` | string | `always` | Quando confirmar: `always`, `high-only`, `never` |
| `f5xc.autoRefreshInterval` | number | `0` | Intervalo de atualização automática em segundos (0 = desativado) |
| `f5xc.viewMode` | string | `console` | Modo de visualização: `console` ou `full` |
| `f5xc.showPreviewApis` | boolean | `true` | Exibir tipos de recursos de API beta/preview |
