## Atalhos Padrão

Os atalhos de teclado podem ser personalizados por meio do editor de Atalhos de Teclado do VS Code (`Ctrl+K Ctrl+S` / `Cmd+K Cmd+S`). Pesquise por "f5xc" para encontrar todos os comandos da extensão.

Todos os mais de 40 comandos estão acessíveis pela Paleta de Comandos (`Ctrl+Shift+P` / `Cmd+Shift+P`) pesquisando por "F5 XC".
