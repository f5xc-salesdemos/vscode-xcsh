A extensão suporta mais de 236 tipos de recursos em 25 domínios de API do F5 XC. Os recursos são organizados por categoria de domínio na visualização em árvore.

## Categorias de Domínio

- **Entrega de Aplicações**: HTTP Load Balancers, TCP Load Balancers, Origin Pools, Health Checks
- **Segurança**: Web App Firewalls, API Security, Bot Defense, DDoS Protection
- **Rede**: Virtual Networks, Sites, Fleet Management, Network Policies
- **DNS**: DNS Zones, DNS Load Balancers, DNS Records
- **Observabilidade**: Dashboards, Alerts, Log Receivers
- **Infraestrutura**: Cloud Sites, Cloud Credentials, Service Policies

A lista completa dos tipos de recursos suportados é gerada automaticamente a partir da especificação OpenAPI do F5 XC e é atualizada automaticamente quando novas versões da API são lançadas.
