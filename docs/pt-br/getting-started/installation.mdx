## VS Code Marketplace

Pesquise por "F5 Distributed Cloud Tools" no painel de Extensões do VS Code (`Ctrl+Shift+X` / `Cmd+Shift+X`) e clique em **Instalar**.

## Instalação Manual via VSIX

1. Baixe o arquivo `.vsix` na página de [GitHub Releases](https://github.com/f5-sales-demo/vscode-f5xc-tools/releases).
2. No VS Code, abra a Paleta de Comandos (`Ctrl+Shift+P` / `Cmd+Shift+P`).
3. Execute **Extensions: Install from VSIX** e selecione o arquivo baixado.

## Requisitos

- VS Code 1.107.0 ou posterior
- Um tenant do F5 Distributed Cloud
- Credenciais de API (certificado ou token de API)
