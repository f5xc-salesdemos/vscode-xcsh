O F5 Distributed Cloud Tools é uma extensão do VS Code que permite navegar, criar, editar e excluir recursos do F5 XC diretamente do seu editor. Ele oferece operações CRUD completas, JSON IntelliSense com validação de schema, ferramentas de observabilidade e monitoramento de status da nuvem para mais de 236 tipos de recursos em 25 domínios de API.
