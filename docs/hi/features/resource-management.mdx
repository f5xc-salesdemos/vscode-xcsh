## संसाधन ब्राउज़ करना

**Resources** ट्री व्यू संसाधनों को डोमेन और श्रेणी के अनुसार व्यवस्थित करके प्रदर्शित करता है। संसाधन प्रकार देखने के लिए किसी श्रेणी को विस्तारित करें, फिर अपने नेमस्पेस में व्यक्तिगत संसाधनों की सूची देखने के लिए किसी प्रकार को विस्तारित करें।

## संसाधन बनाना

1. ट्री में किसी संसाधन प्रकार पर राइट-क्लिक करें और **Create** चुनें।
2. एक स्कीमा-मान्य टेम्पलेट के साथ JSON संपादक खुलता है।
3. आवश्यक फ़ील्ड भरें (IntelliSense स्वतः-पूर्णता प्रदान करता है)।
4. F5 XC API को संसाधन सबमिट करने के लिए फ़ाइल सहेजें।

## संसाधन संपादित करना

1. संपादक में खोलने के लिए ट्री में किसी संसाधन पर क्लिक करें।
2. पूर्ण IntelliSense समर्थन के साथ फ़ील्ड संशोधित करें।
3. API में परिवर्तन भेजने के लिए सहेजें।

## संसाधन हटाना

1. किसी संसाधन पर राइट-क्लिक करें और **Delete** चुनें।
2. हटाने की पुष्टि करें (`f5xc.confirmDelete` और `f5xc.deleteConfirmationLevel` के माध्यम से कॉन्फ़िगर करने योग्य)।
