## स्थिति निगरानी

**क्लाउड स्थिति** ट्री व्यू F5 Distributed Cloud सेवाओं की वर्तमान स्वास्थ्य स्थिति प्रदर्शित करता है। स्थिति आधिकारिक F5 स्टेटस पेज से प्राप्त की जाती है और निम्नलिखित दिखाती है:

- समग्र प्लेटफ़ॉर्म स्थिति
- प्रति-घटक स्वास्थ्य (API, Console, Data Plane, आदि)
- सक्रिय घटनाएँ और रखरखाव विंडो

यह व्यू कॉन्फ़िगर किए गए रिफ्रेश अंतराल (`f5xc.autoRefreshInterval`) के आधार पर स्वचालित रूप से अपडेट होता है।
