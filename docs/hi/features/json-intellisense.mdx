## स्कीमा सत्यापन

यह एक्सटेंशन सामान्य संसाधन प्रकारों के लिए JSON स्कीमा प्रदान करता है। संसाधन JSON फ़ाइलों को संपादित करते समय, VS Code आपके कॉन्फ़िगरेशन को रियल-टाइम में सत्यापित करता है और त्रुटियों को हाइलाइट करता है।

## समर्थित स्कीमा

- HTTP Load Balancer (`.http_loadbalancer.json`)
- Origin Pool (`.origin_pool.json`)
- App Firewall / WAF (`.app_firewall.json`)
- Health Check (`.healthcheck.json`)
- TCP Load Balancer (`.tcp_loadbalancer.json`)
- DNS Load Balancer (`.dns_load_balancer.json`)
- Virtual Site (`.virtual_site.json`)

## ऑटो-कम्प्लीशन

IntelliSense, F5 XC OpenAPI स्पेक के आधार पर फ़ील्ड नामों, enum मानों और नेस्टेड ऑब्जेक्ट संरचनाओं के लिए ऑटो-कम्प्लीशन प्रदान करता है।
