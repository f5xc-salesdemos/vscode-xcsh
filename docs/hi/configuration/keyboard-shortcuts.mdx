## डिफ़ॉल्ट कीबाइंडिंग

कीबोर्ड शॉर्टकट को VS Code के Keyboard Shortcuts एडिटर (`Ctrl+K Ctrl+S` / `Cmd+K Cmd+S`) के माध्यम से अनुकूलित किया जा सकता है। सभी एक्सटेंशन कमांड खोजने के लिए "f5xc" खोजें।

सभी 40+ कमांड Command Palette (`Ctrl+Shift+P` / `Cmd+Shift+P`) के माध्यम से "F5 XC" खोजकर उपलब्ध हैं।
