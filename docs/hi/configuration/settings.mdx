## उपलब्ध सेटिंग्स

| सेटिंग | प्रकार | डिफ़ॉल्ट | विवरण |
|---------|------|---------|-------------|
| `f5xc.logLevel` | string | `info` | लॉग वर्बोसिटी: `debug`, `info`, `warn`, `error` |
| `f5xc.defaultNamespace` | string | `default` | रिसोर्स ऑपरेशन के लिए डिफ़ॉल्ट नेमस्पेस |
| `f5xc.confirmDelete` | boolean | `true` | रिसोर्स हटाने से पहले पुष्टिकरण डायलॉग दिखाएं |
| `f5xc.deleteConfirmationLevel` | string | `always` | कब पुष्टि करें: `always`, `high-only`, `never` |
| `f5xc.autoRefreshInterval` | number | `0` | ऑटो-रिफ़्रेश अंतराल सेकंड में (0 = अक्षम) |
| `f5xc.viewMode` | string | `console` | व्यू मोड: `console` या `full` |
| `f5xc.showPreviewApis` | boolean | `true` | बीटा/प्रीव्यू API रिसोर्स प्रकार दिखाएं |
