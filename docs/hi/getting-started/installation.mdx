## VS Code मार्केटप्लेस

VS Code एक्सटेंशन पैनल (`Ctrl+Shift+X` / `Cmd+Shift+X`) में "F5 Distributed Cloud Tools" खोजें और **इंस्टॉल** पर क्लिक करें।

## मैनुअल VSIX इंस्टॉल

1. [GitHub Releases](https://github.com/f5-sales-demo/vscode-f5xc-tools/releases) पेज से `.vsix` फ़ाइल डाउनलोड करें।
2. VS Code में, कमांड पैलेट खोलें (`Ctrl+Shift+P` / `Cmd+Shift+P`)।
3. **Extensions: Install from VSIX** चलाएँ और डाउनलोड की गई फ़ाइल चुनें।

## आवश्यकताएँ

- VS Code 1.107.0 या बाद का संस्करण
- एक F5 Distributed Cloud टेनेंट
- API क्रेडेंशियल्स (सर्टिफिकेट या API टोकन)
