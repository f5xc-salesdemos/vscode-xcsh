## प्रोफ़ाइल जोड़ना

1. F5 XC साइडबार में **Profiles** व्यू खोलें।
2. एक नई प्रोफ़ाइल जोड़ने के लिए **+** आइकन पर क्लिक करें।
3. अपना टेनेंट URL दर्ज करें (उदा., `https://your-tenant.console.ves.volterra.io`)।
4. एक प्रमाणीकरण विधि चुनें:
   - **API Token**: F5 XC कंसोल से अपना API टोकन पेस्ट करें।
   - **P12 Certificate**: अपनी `.p12` सर्टिफ़िकेट फ़ाइल चुनें और पासवर्ड दर्ज करें।
5. एक डिफ़ॉल्ट नेमस्पेस सेट करें।

## प्रोफ़ाइल प्रबंधन

- **प्रोफ़ाइल स्विच करें**: किसी प्रोफ़ाइल को सक्रिय करने के लिए Profiles व्यू में उस पर क्लिक करें।
- **संपादित करें**: किसी प्रोफ़ाइल पर राइट-क्लिक करें और **Edit Profile** चुनें।
- **हटाएँ**: किसी प्रोफ़ाइल पर राइट-क्लिक करें और **Delete Profile** चुनें।

प्रोफ़ाइल VS Code के बिल्ट-इन सीक्रेट स्टोरेज (Linux पर XDG-अनुरूप) का उपयोग करके सुरक्षित रूप से संग्रहीत की जाती हैं।
