यह एक्सटेंशन 25 F5 XC API डोमेन में 236+ संसाधन प्रकारों का समर्थन करता है। ट्री व्यू में संसाधनों को डोमेन श्रेणी के अनुसार व्यवस्थित किया गया है।

## डोमेन श्रेणियाँ

- **एप्लिकेशन डिलीवरी**: HTTP Load Balancers, TCP Load Balancers, Origin Pools, Health Checks
- **सुरक्षा**: Web App Firewalls, API Security, Bot Defense, DDoS Protection
- **नेटवर्किंग**: Virtual Networks, Sites, Fleet Management, Network Policies
- **DNS**: DNS Zones, DNS Load Balancers, DNS Records
- **अवलोकनीयता**: Dashboards, Alerts, Log Receivers
- **अवसंरचना**: Cloud Sites, Cloud Credentials, Service Policies

समर्थित संसाधन प्रकारों की पूर्ण सूची F5 XC OpenAPI विनिर्देश से स्वचालित रूप से उत्पन्न होती है और नए API संस्करण जारी होने पर स्वचालित रूप से अपडेट हो जाती है।
