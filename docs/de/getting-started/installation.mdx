## VS Code Marketplace

Suchen Sie im VS Code-Erweiterungspanel (`Ctrl+Shift+X` / `Cmd+Shift+X`) nach "F5 Distributed Cloud Tools" und klicken Sie auf **Installieren**.

## Manuelle VSIX-Installation

1. Laden Sie die `.vsix`-Datei von der [GitHub Releases](https://github.com/f5xc-salesdemos/vscode-f5xc-tools/releases)-Seite herunter.
2. Öffnen Sie in VS Code die Befehlspalette (`Ctrl+Shift+P` / `Cmd+Shift+P`).
3. Führen Sie **Extensions: Install from VSIX** aus und wählen Sie die heruntergeladene Datei aus.

## Voraussetzungen

- VS Code 1.107.0 oder höher
- Ein F5 Distributed Cloud-Mandant
- API-Anmeldeinformationen (Zertifikat oder API-Token)
