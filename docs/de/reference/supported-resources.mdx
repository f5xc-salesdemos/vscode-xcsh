Die Erweiterung unterstützt über 236 Ressourcentypen in 25 F5 XC API-Domänen. Ressourcen werden in der Baumansicht nach Domänenkategorie organisiert.

## Domänenkategorien

- **Application Delivery**: HTTP Load Balancers, TCP Load Balancers, Origin Pools, Health Checks
- **Sicherheit**: Web App Firewalls, API Security, Bot Defense, DDoS Protection
- **Netzwerk**: Virtual Networks, Sites, Fleet Management, Network Policies
- **DNS**: DNS Zones, DNS Load Balancers, DNS Records
- **Observability**: Dashboards, Alerts, Log Receivers
- **Infrastruktur**: Cloud Sites, Cloud Credentials, Service Policies

Die vollständige Liste der unterstützten Ressourcentypen wird automatisch aus der F5 XC OpenAPI-Spezifikation generiert und aktualisiert sich automatisch, wenn neue API-Versionen veröffentlicht werden.
