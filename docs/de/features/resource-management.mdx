## Ressourcen durchsuchen

Die Baumansicht **Ressourcen** zeigt Ressourcen organisiert nach Domäne und Kategorie an. Erweitern Sie eine Kategorie, um Ressourcentypen anzuzeigen, und erweitern Sie dann einen Typ, um einzelne Ressourcen in Ihrem Namespace aufzulisten.

## Ressourcen erstellen

1. Klicken Sie mit der rechten Maustaste auf einen Ressourcentyp in der Baumansicht und wählen Sie **Erstellen**.
2. Ein JSON-Editor öffnet sich mit einer schema-validierten Vorlage.
3. Füllen Sie die erforderlichen Felder aus (IntelliSense bietet automatische Vervollständigung).
4. Speichern Sie die Datei, um die Ressource an die F5 XC API zu übermitteln.

## Ressourcen bearbeiten

1. Klicken Sie auf eine Ressource in der Baumansicht, um sie im Editor zu öffnen.
2. Ändern Sie Felder mit vollständiger IntelliSense-Unterstützung.
3. Speichern Sie, um die Änderungen an die API zu übertragen.

## Ressourcen löschen

1. Klicken Sie mit der rechten Maustaste auf eine Ressource und wählen Sie **Löschen**.
2. Bestätigen Sie die Löschung (konfigurierbar über `f5xc.confirmDelete` und `f5xc.deleteConfirmationLevel`).
