## Statusüberwachung

Die Baumansicht **Cloud-Status** zeigt den aktuellen Zustand der F5 Distributed Cloud-Dienste an. Der Status wird von der offiziellen F5-Statusseite abgerufen und zeigt:

- Gesamtstatus der Plattform
- Zustand einzelner Komponenten (API, Konsole, Data Plane usw.)
- Aktive Vorfälle und Wartungsfenster

Die Ansicht wird automatisch basierend auf dem konfigurierten Aktualisierungsintervall (`f5xc.autoRefreshInterval`) aktualisiert.
