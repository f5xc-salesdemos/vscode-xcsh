F5 Distributed Cloud Tools ist eine VS Code-Erweiterung, mit der Sie F5 XC-Ressourcen direkt in Ihrem Editor durchsuchen, erstellen, bearbeiten und löschen können. Sie bietet vollständige CRUD-Operationen, JSON IntelliSense mit Schema-Validierung, Observability-Tools und Cloud-Statusüberwachung für über 236 Ressourcentypen in 25 API-Domänen.
