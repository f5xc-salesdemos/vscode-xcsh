## Standard-Tastenzuweisungen

Tastaturkürzel können über den Tastaturkürzel-Editor von VS Code (`Ctrl+K Ctrl+S` / `Cmd+K Cmd+S`) angepasst werden. Suchen Sie nach "f5xc", um alle Erweiterungsbefehle zu finden.

Alle über 40 Befehle sind über die Befehlspalette (`Ctrl+Shift+P` / `Cmd+Shift+P`) zugänglich, indem Sie nach "F5 XC" suchen.
