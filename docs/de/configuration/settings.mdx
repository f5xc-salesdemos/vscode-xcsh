## Verfügbare Einstellungen

| Einstellung | Typ | Standard | Beschreibung |
|-------------|-----|----------|--------------|
| `f5xc.logLevel` | string | `info` | Log-Ausführlichkeit: `debug`, `info`, `warn`, `error` |
| `f5xc.defaultNamespace` | string | `default` | Standard-Namespace für Ressourcenoperationen |
| `f5xc.confirmDelete` | boolean | `true` | Bestätigungsdialog vor dem Löschen von Ressourcen anzeigen |
| `f5xc.deleteConfirmationLevel` | string | `always` | Wann bestätigt werden soll: `always`, `high-only`, `never` |
| `f5xc.autoRefreshInterval` | number | `0` | Automatisches Aktualisierungsintervall in Sekunden (0 = deaktiviert) |
| `f5xc.viewMode` | string | `console` | Anzeigemodus: `console` oder `full` |
| `f5xc.showPreviewApis` | boolean | `true` | Beta-/Vorschau-API-Ressourcentypen anzeigen |
