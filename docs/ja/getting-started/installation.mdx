## VS Code マーケットプレイス

VS Code の拡張機能パネル（`Ctrl+Shift+X` / `Cmd+Shift+X`）で「F5 Distributed Cloud Tools」を検索し、**インストール** をクリックします。

## 手動 VSIX インストール

1. [GitHub Releases](https://github.com/f5-sales-demo/vscode-f5xc-tools/releases) ページから `.vsix` ファイルをダウンロードします。
2. VS Code でコマンドパレット（`Ctrl+Shift+P` / `Cmd+Shift+P`）を開きます。
3. **Extensions: Install from VSIX** を実行し、ダウンロードしたファイルを選択します。

## 要件

- VS Code 1.107.0 以降
- F5 Distributed Cloud テナント
- API 資格情報（証明書または API トークン）
