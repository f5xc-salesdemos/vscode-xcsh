## プロファイルの追加

1. F5 XC サイドバーで **Profiles** ビューを開きます。
2. **+** アイコンをクリックして新しいプロファイルを追加します。
3. テナント URL を入力します（例: `https://your-tenant.console.ves.volterra.io`）。
4. 認証方法を選択します:
   - **API トークン**: F5 XC コンソールから取得した API トークンを貼り付けます。
   - **P12 証明書**: `.p12` 証明書ファイルを選択し、パスワードを入力します。
5. デフォルトのネームスペースを設定します。

## プロファイルの管理

- **プロファイルの切り替え**: Profiles ビューでプロファイルをクリックしてアクティブにします。
- **編集**: プロファイルを右クリックして **Edit Profile** を選択します。
- **削除**: プロファイルを右クリックして **Delete Profile** を選択します。

プロファイルは VS Code の組み込みシークレットストレージ（Linux では XDG 準拠）を使用して安全に保存されます。
