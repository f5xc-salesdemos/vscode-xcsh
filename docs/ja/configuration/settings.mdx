## 利用可能な設定

| 設定 | 型 | デフォルト | 説明 |
|---------|------|---------|-------------|
| `f5xc.logLevel` | string | `info` | ログの詳細度: `debug`, `info`, `warn`, `error` |
| `f5xc.defaultNamespace` | string | `default` | リソース操作のデフォルト名前空間 |
| `f5xc.confirmDelete` | boolean | `true` | リソース削除前に確認ダイアログを表示する |
| `f5xc.deleteConfirmationLevel` | string | `always` | 確認のタイミング: `always`, `high-only`, `never` |
| `f5xc.autoRefreshInterval` | number | `0` | 自動更新間隔（秒単位、0 = 無効） |
| `f5xc.viewMode` | string | `console` | 表示モード: `console` または `full` |
| `f5xc.showPreviewApis` | boolean | `true` | ベータ/プレビュー API リソースタイプを表示する |
