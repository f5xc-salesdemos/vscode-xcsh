## デフォルトのキーバインド

キーボードショートカットは、VS Code のキーボードショートカットエディター（`Ctrl+K Ctrl+S` / `Cmd+K Cmd+S`）でカスタマイズできます。「f5xc」で検索すると、すべての拡張機能コマンドが表示されます。

40以上のすべてのコマンドは、コマンドパレット（`Ctrl+Shift+P` / `Cmd+Shift+P`）から「F5 XC」と検索してアクセスできます。
