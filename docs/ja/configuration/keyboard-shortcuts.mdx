## デフォルトのキーバインド

キーボードショートカットは、VS Codeのキーボードショートカットエディター（`Ctrl+K Ctrl+S` / `Cmd+K Cmd+S`）からカスタマイズできます。「f5xc」で検索すると、すべての拡張機能コマンドを見つけることができます。

40以上のすべてのコマンドは、コマンドパレット（`Ctrl+Shift+P` / `Cmd+Shift+P`）から「F5 XC」と検索してアクセスできます。
