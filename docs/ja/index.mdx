F5 Distributed Cloud Tools は、エディターから直接 F5 XC リソースの参照、作成、編集、削除を可能にする VS Code 拡張機能です。25 の API ドメインにわたる 236 以上のリソースタイプに対して、完全な CRUD 操作、スキーマバリデーション付き JSON IntelliSense、オブザーバビリティツール、およびクラウドステータスモニタリングを提供します。
