この拡張機能は、25 の F5 XC API ドメインにわたる 236 以上のリソースタイプをサポートしています。リソースはツリービューでドメインカテゴリ別に整理されています。

## ドメインカテゴリ

- **アプリケーションデリバリー**: HTTP ロードバランサー、TCP ロードバランサー、オリジンプール、ヘルスチェック
- **セキュリティ**: Web アプリケーションファイアウォール、API セキュリティ、ボット防御、DDoS 保護
- **ネットワーキング**: 仮想ネットワーク、サイト、フリート管理、ネットワークポリシー
- **DNS**: DNS ゾーン、DNS ロードバランサー、DNS レコード
- **オブザーバビリティ**: ダッシュボード、アラート、ログレシーバー
- **インフラストラクチャ**: クラウドサイト、クラウド認証情報、サービスポリシー

サポートされるリソースタイプの完全なリストは、F5 XC OpenAPI 仕様から自動生成され、新しい API バージョンがリリースされると自動的に更新されます。
