この拡張機能は、25のF5 XC APIドメインにわたる236以上のリソースタイプをサポートしています。リソースはツリービューでドメインカテゴリ別に整理されています。

## ドメインカテゴリ

- **アプリケーションデリバリー**: HTTP Load Balancers、TCP Load Balancers、Origin Pools、Health Checks
- **セキュリティ**: Web App Firewalls、API Security、Bot Defense、DDoS Protection
- **ネットワーキング**: Virtual Networks、Sites、Fleet Management、Network Policies
- **DNS**: DNS Zones、DNS Load Balancers、DNS Records
- **オブザーバビリティ**: Dashboards、Alerts、Log Receivers
- **インフラストラクチャ**: Cloud Sites、Cloud Credentials、Service Policies

サポートされるリソースタイプの完全なリストは、F5 XC OpenAPI仕様から自動生成され、新しいAPIバージョンがリリースされると自動的に更新されます。
