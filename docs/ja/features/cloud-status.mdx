## ステータス監視

**クラウドステータス**ツリービューは、F5 Distributed Cloud サービスの現在の正常性を表示します。ステータスは F5 の公式ステータスページから取得され、以下の情報が表示されます：

- プラットフォーム全体のステータス
- コンポーネントごとの正常性（API、コンソール、データプレーンなど）
- アクティブなインシデントおよびメンテナンスウィンドウ

ビューは設定された更新間隔（`f5xc.autoRefreshInterval`）に基づいて自動的に更新されます。
