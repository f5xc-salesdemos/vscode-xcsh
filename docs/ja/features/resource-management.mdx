## リソースの閲覧

**Resources** ツリービューには、ドメインとカテゴリ別に整理されたリソースが表示されます。カテゴリを展開するとリソースタイプが表示され、さらにタイプを展開すると名前空間内の個々のリソースが一覧表示されます。

## リソースの作成

1. ツリー内のリソースタイプを右クリックし、**Create** を選択します。
2. スキーマ検証済みのテンプレートを含む JSON エディタが開きます。
3. 必須フィールドを入力します（IntelliSense による自動補完が利用できます）。
4. ファイルを保存すると、リソースが F5 XC API に送信されます。

## リソースの編集

1. ツリー内のリソースをクリックしてエディタで開きます。
2. 完全な IntelliSense サポートを利用してフィールドを変更します。
3. 保存すると変更が API にプッシュされます。

## リソースの削除

1. リソースを右クリックし、**Delete** を選択します。
2. 削除を確認します（`f5xc.confirmDelete` および `f5xc.deleteConfirmationLevel` で設定可能です）。
