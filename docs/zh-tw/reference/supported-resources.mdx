此擴充功能支援跨 25 個 F5 XC API 網域的 236 種以上資源類型。資源在樹狀檢視中按網域類別進行組織。

## 網域類別

- **應用程式交付**：HTTP 負載平衡器、TCP 負載平衡器、來源池、健康檢查
- **安全性**：Web 應用程式防火牆、API 安全性、Bot 防禦、DDoS 防護
- **網路**：虛擬網路、站點、車隊管理、網路原則
- **DNS**：DNS 區域、DNS 負載平衡器、DNS 記錄
- **可觀測性**：儀表板、警報、日誌接收器
- **基礎架構**：雲端站點、雲端憑證、服務原則

支援的資源類型完整清單是從 F5 XC OpenAPI 規格自動產生的，並會在新的 API 版本發布時自動更新。
