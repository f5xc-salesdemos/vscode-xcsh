## VS Code Marketplace

在 VS Code 擴充功能面板（`Ctrl+Shift+X` / `Cmd+Shift+X`）中搜尋「F5 Distributed Cloud Tools」，然後點擊**安裝**。

## 手動 VSIX 安裝

1. 從 [GitHub Releases](https://github.com/f5-sales-demo/vscode-f5xc-tools/releases) 頁面下載 `.vsix` 檔案。
2. 在 VS Code 中，開啟命令面板（`Ctrl+Shift+P` / `Cmd+Shift+P`）。
3. 執行 **Extensions: Install from VSIX** 並選擇已下載的檔案。

## 需求

- VS Code 1.107.0 或更新版本
- 一個 F5 Distributed Cloud 租戶
- API 憑證（憑證或 API 權杖）
