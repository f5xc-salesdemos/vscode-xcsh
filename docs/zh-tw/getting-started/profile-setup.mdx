## 新增設定檔

1. 在 F5 XC 側邊欄中開啟 **Profiles** 檢視。
2. 點擊 **+** 圖示以新增設定檔。
3. 輸入您的租戶 URL（例如 `https://your-tenant.console.ves.volterra.io`）。
4. 選擇驗證方式：
   - **API Token**：從 F5 XC 主控台貼上您的 API 權杖。
   - **P12 Certificate**：選擇您的 `.p12` 憑證檔案並輸入密碼。
5. 設定預設命名空間。

## 管理設定檔

- **切換設定檔**：在 Profiles 檢視中點擊某個設定檔以將其設為使用中。
- **編輯**：右鍵點擊設定檔並選擇 **Edit Profile**。
- **刪除**：右鍵點擊設定檔並選擇 **Delete Profile**。

設定檔使用 VS Code 內建的密鑰儲存機制（在 Linux 上符合 XDG 規範）安全地儲存。
