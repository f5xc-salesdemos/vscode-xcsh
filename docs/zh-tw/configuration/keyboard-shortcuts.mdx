## 預設按鍵綁定

鍵盤快速鍵可透過 VS Code 的鍵盤快速鍵編輯器（`Ctrl+K Ctrl+S` / `Cmd+K Cmd+S`）進行自訂。搜尋 "f5xc" 即可找到所有擴充功能命令。

所有 40 多個命令皆可透過命令面板（`Ctrl+Shift+P` / `Cmd+Shift+P`）搜尋 "F5 XC" 來存取。
