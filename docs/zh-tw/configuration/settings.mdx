## 可用設定

| 設定 | 類型 | 預設值 | 說明 |
|---------|------|---------|-------------|
| `f5xc.logLevel` | string | `info` | 日誌詳細程度：`debug`、`info`、`warn`、`error` |
| `f5xc.defaultNamespace` | string | `default` | 資源操作的預設命名空間 |
| `f5xc.confirmDelete` | boolean | `true` | 刪除資源前顯示確認對話框 |
| `f5xc.deleteConfirmationLevel` | string | `always` | 確認時機：`always`、`high-only`、`never` |
| `f5xc.autoRefreshInterval` | number | `0` | 自動重新整理間隔（秒）（0 = 停用） |
| `f5xc.viewMode` | string | `console` | 檢視模式：`console` 或 `full` |
| `f5xc.showPreviewApis` | boolean | `true` | 顯示 Beta/預覽版 API 資源類型 |
