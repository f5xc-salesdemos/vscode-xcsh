## 瀏覽資源

**Resources** 樹狀檢視會依網域和類別組織顯示資源。展開類別可查看資源類型，然後展開類型即可列出您命名空間中的個別資源。

## 建立資源

1. 在樹狀檢視中的資源類型上按右鍵，選擇 **Create**。
2. 會開啟一個經過結構描述驗證的 JSON 編輯器範本。
3. 填寫必填欄位（IntelliSense 提供自動完成功能）。
4. 儲存檔案以將資源提交至 F5 XC API。

## 編輯資源

1. 在樹狀檢視中點擊資源，即可在編輯器中開啟。
2. 透過完整的 IntelliSense 支援修改欄位。
3. 儲存以將變更推送至 API。

## 刪除資源

1. 在資源上按右鍵，選擇 **Delete**。
2. 確認刪除（可透過 `f5xc.confirmDelete` 和 `f5xc.deleteConfirmationLevel` 進行設定）。
