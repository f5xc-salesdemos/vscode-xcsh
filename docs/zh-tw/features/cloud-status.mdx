## 狀態監控

**雲端狀態**樹狀檢視會顯示 F5 Distributed Cloud 服務的目前健康狀態。狀態資訊從官方 F5 狀態頁面擷取，並顯示：

- 整體平台狀態
- 各元件健康狀態（API、Console、Data Plane 等）
- 進行中的事件與維護時段

此檢視會根據設定的重新整理間隔（`f5xc.autoRefreshInterval`）自動更新。
