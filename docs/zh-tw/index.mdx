F5 Distributed Cloud Tools 是一個 VS Code 擴充功能，讓您能夠直接從編輯器中瀏覽、建立、編輯和刪除 F5 XC 資源。它提供完整的 CRUD 操作、具有結構驗證的 JSON IntelliSense、可觀測性工具，以及涵蓋 25 個 API 網域中 236 種以上資源類型的雲端狀態監控功能。
